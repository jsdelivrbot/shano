Beaufix (Install)
=================

This repository contains the base theme for Obelix 8 installation process.

Installation
------------

`composer require world-vision/beaufix_install:1.x-dev`
 
Dependencies
------------

* [drupal/core](http://drupal.org/project/drupal)
